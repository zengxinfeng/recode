# 物品记录管理器

一个基于PySide6的物品记录管理应用，用于记录和管理个人物品的购买信息，计算物品的使用成本和统计数据。

## 功能特点

- 添加、修改、删除物品记录
- 自动计算物品的已使用天数和日均使用价格
- 统计总资产、日均总成本和物品数量
- 支持按物品名称搜索
- 数据自动保存到本地文件

## 安装方法

1. 克隆项目到本地
2. 安装依赖包：
   ```
   pip install -r requirements.txt
   ```
3. 运行应用：
   ```
   python app/main.py
   ```

## 技术栈

- Python 3.9+
- PySide6 6.6.0

## 运行环境

- Windows、macOS、Linux
- Python 3.9 或更高版本

## 使用说明

1. 启动应用后，在"添加物品"区域输入物品名称、购买价格和购买日期
2. 点击"添加物品"按钮将物品添加到列表
3. 在"物品列表"中查看所有物品的详细信息
4. 点击物品右侧的"修改"按钮可以修改物品信息
5. 点击"删除"按钮可以删除物品
6. 在"搜索"区域输入关键词可以搜索物品
7. 顶部的"统计信息"区域会实时显示总资产、日均总成本和物品数量

## 数据存储

应用会将数据自动保存到 `app/resources/datas/items.json` 文件中，确保数据不会丢失。
